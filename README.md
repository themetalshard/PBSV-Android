# PB Schedule Viewer (Android port)
PB Schedule Viewer is an app for viewing schedules for the Roblox group Pinewood Builders.

It relies on the Pinewood Builders server rehosted on my website. (https://pbsv.themetalshard.space)
The server source code will be available later.

Currently this app is a bit of a mess in the code, if someone wants to fix it, feel free to.
